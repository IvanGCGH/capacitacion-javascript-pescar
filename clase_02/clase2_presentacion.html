üí° Importante: Todos estos m√©todos reciben una funci√≥n como par√°metro (callback)

üìå Nota: forEach NO retorna nada, solo ejecuta c√≥digo para cada elemento

map - Transformar elementos

const numeros = [1, 2, 3, 4, 5]; // map crea un NUEVO array con los resultados const duplicados = numeros.map((num) => num * 2); console.log(duplicados); // [2, 4, 6, 8, 10] console.log(numeros); // [1, 2, 3, 4, 5] (sin cambios) // Transformar objetos const personas = [ { nombre: "Ana", edad: 25 }, { nombre: "Juan", edad: 30 } ]; const nombres = personas.map((persona) => persona.nombre); console.log(nombres); // ["Ana", "Juan"] const conSaludo = personas.map((p) => `Hola ${p.nombre}`); console.log(conSaludo); // ["Hola Ana", "Hola Juan"]
üí° Tip: Us√° map cuando quieras transformar cada elemento y obtener un nuevo array

filter - Filtrar elementos

üí° Tip: filter retorna todos los elementos que hacen que la funci√≥n retorne true

find - Encontrar un elemento

reduce - Reducir a un valor

const numeros = [1, 2, 3, 4, 5]; // reduce combina todos los elementos en un √∫nico valor const suma = numeros.reduce((acumulador, numero) => { return acumulador + numero; }, 0); // 0 es el valor inicial del acumulador console.log(suma); // 15 // Forma corta const sumaCorta = numeros.reduce((acc, num) => acc + num, 0); // Encontrar el m√°ximo const maximo = numeros.reduce((max, num) => { return num > max ? num : max; }, numeros[0]); console.log(maximo); // 5 // Contar ocurrencias const frutas = ["manzana", "banana", "manzana", "naranja", "banana"]; const conteo = frutas.reduce((acc, fruta) => { acc[fruta] = (acc[fruta] || 0) + 1; return acc; }, {}); console.log(conteo); // { manzana: 2, banana: 2, naranja: 1 }

üîó Encadenando M√©todos (Chaining)

Destructuring (Desestructuraci√≥n)

Forma moderna de extraer valores de arrays y objetos

Destructuring de Arrays

const numeros = [1, 2, 3, 4, 5]; // Sin destructuring const primero = numeros[0]; const segundo = numeros[1]; // Con destructuring const [a, b, c] = numeros; console.log(a); // 1 console.log(b); // 2 console.log(c); // 3 // Saltear elementos const [x, , z] = numeros; // Saltea el segundo console.log(x); // 1 console.log(z); // 3 // Rest operator const [primero, ...resto] = numeros; console.log(primero); // 1 console.log(resto); // [2, 3, 4, 5]

Destructuring de Objetos

const persona = { nombre: "Ana", edad: 25, ciudad: "Buenos Aires", profesion: "Desarrolladora" }; // Sin destructuring const nombre = persona.nombre; const edad = persona.edad; // Con destructuring const { nombre, edad } = persona; console.log(nombre); // "Ana" console.log(edad); // 25 // Renombrar variables const { nombre: nombrePersona, edad: a√±os } = persona; console.log(nombrePersona); // "Ana" console.log(a√±os); // 25 // Valores por defecto const { nombre, pais = "Argentina" } = persona; console.log(pais); // "Argentina" // Rest operator const { nombre, ...restoInfo } = persona; console.log(restoInfo); // { edad: 25, ciudad: "Buenos Aires", profesion: "Desarrolladora" }

JSON (JavaScript Object Notation)

Formato est√°ndar para intercambiar datos entre cliente y servidor

const persona = { nombre: "Ana", edad: 25, hobbies: ["leer", "programar", "viajar"] }; // Convertir objeto a JSON (string) const jsonString = JSON.stringify(persona); console.log(jsonString); // '{"nombre":"Ana","edad":25,"hobbies":["leer","programar","viajar"]}' console.log(typeof jsonString); // "string" // Convertir JSON a objeto const objetoRecuperado = JSON.parse(jsonString); console.log(objetoRecuperado); // { nombre: "Ana", edad: 25, hobbies: ["leer", "programar", "viajar"] } console.log(typeof objetoRecuperado); // "object" // JSON.stringify con formato bonito const jsonBonito = JSON.stringify(persona, null, 2); console.log(jsonBonito);

Spread Operator (...)

Operador para "expandir" arrays y objetos

// Spread con Arrays const arr1 = [1, 2, 3]; const arr2 = [4, 5, 6]; const combinado = [...arr1, ...arr2]; console.log(combinado); // [1, 2, 3, 4, 5, 6] // Copiar array const copia = [...arr1]; console.log(copia); // [1, 2, 3] // Spread con Objetos const persona = { nombre: "Ana", edad: 25 }; const direccion = { ciudad: "Buenos Aires", pais: "Argentina" }; const personaCompleta = { ...persona, ...direccion }; console.log(personaCompleta); // { nombre: "Ana", edad: 25, ciudad: "Buenos Aires", pais: "Argentina" } // Sobrescribir propiedades const actualizado = { ...persona, edad: 26, profesion: "Dev" }; console.log(actualizado); // { nombre: "Ana", edad: 26, profesion: "Dev" }
